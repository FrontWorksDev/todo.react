# Todo.app (React.js)

Reactで作成されたタスク管理アプリです。

BackEndのリポジトリは[こちら](https://github.com/FrontWorksDev/todo.go)

- FrontEnd

  - Material UI v5.2.7
- BackEnd
  - Golang v17.5.0
  - Gin
  - GORM
- Test
  - React Testing Library

- Create Task

- Delete Task
